# AI-Technologies-Power-Bi-Insights-Project-
Power BI dashboard analyzing global AI and technology companies, focusing on market performance, revenue efficiency, R&amp;D investment, and AI innovation through patent and focus-area analysis.
Using dataste from Kaggle, this project presents an exploratory Power BI dashboard analyzing global AI and technology companies, with a focus on market performance, operational efficiency, R&D investment, and AI innovation.
The goal is to demonstrate how raw industry data can be transformed into clear, decision-oriented insights using data visualization and business intelligence techniques.
The link can be found here: https://www.kaggle.com/datasets/imaadmahmood/tech-and-ai-companies-market-data-2024
